# 🛒 E-Commerce Recommendation Engine (CLI)

A terminal-based e-commerce system built using Python that simulates real-world features like product search, recommendation engine, cart system, and purchase flow.

---

## 🚀 Features

- 🔍 Search Products (by name/category)
- ⭐ Recommendation Engine (based on preference + popularity)
- 🛒 Add to Cart / Remove from Cart
- 💳 Checkout System
- 📜 Search & Purchase History
- ⚙️ Change User Preferences

---

## 🧠 DSA Concepts Used

- Sorting (Top recommendations)
- HashMap (Popularity tracking)
- List Filtering (Search system)
- Basic Scoring Algorithm

---

## 📸 Demo Flow

1. Search → Add to Cart  
2. View Cart → Checkout  
3. Get Recommendations  

---

## 🛠️ Tech Stack

- Python 3
- CLI (Terminal-based UI)

---

## ▶️ How to Run

```bash
python main.py
```
